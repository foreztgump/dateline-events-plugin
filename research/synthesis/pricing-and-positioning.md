### 2.1 EventON (AshanJay Designs / AJDE)

**Positioning:** "An Elegant Event Calendar that meets functionality for WordPress." A premium WordPress event calendar plugin sold as a one-time purchase with an extensive add-on ecosystem. **Pricing Model — Hybrid (Core one-time + Addon subscriptions):**

EventON uses a deliberately split pricing strategy:

| Product | Price | Model |
|---------|-------|-------|
| EventON Lite (Free) | Free | GPL on WordPress.org |

| Addons (each) | $25–$150/yr | Annual subscription per addon |

**Key insight:** The core plugin is deliberately cheap ($40 one-time) — it's a loss leader. The real revenue comes from addon subscriptions ($25–$150/year each). A fully-loaded EventON site paying for the top addons annually can cost **$500+/year** in subscriptions.

## 4. Free vs Pro Splits

### 4.1 EventON (Lite vs Full)

**EventON Lite (free, WordPress.org):**
- Basic event creation and display
- Month accordion and list views
- Event categories and tags
- Location and organizer support
- iCal/Google Calendar export
- Schema.org JSON-LD
- Gutenberg blocks

**EventON Full ($40 one-time) adds:**
- All Lite features (minus restrictions)
- Virtual events (Zoom/Jitsi)
- Webhook support
- Visual designer
- Custom CSS/JS
- Multi-data types
- REST API access
- Language/translation overrides
- PayPal payment basics
- **Ability to use add-ons** (the real gate)

**Key insight:** The main gate is not features — it's the **add-on ecosystem**. EventON Lite can display events adequately, but to sell tickets, collect RSVPs, or add seat maps, you need Full + add-ons. The $40 core purchase is deliberately low-friction to enable the $300–800/yr add-on revenue stream.

### 4.2 The Events Calendar (Free vs Pro)

**TEC Free:**
- Month, list, and day views
- Categories and tags
- Fully responsive
- Google Maps integration
- Gutenberg blocks

**TEC Pro ($199/yr) adds:**
- Week, Photo, and Map views
- Recurring events
- Custom event fields
- Location search
- Advanced widgets
- Elementor compatibility
- Premium support
- Shortcodes
- Event Aggregator import

**Key insight:** TEC's free version has the largest install base of any event calendar. Recurring events is the #1 upgrade driver. Unlike EventON, the free version is genuinely useful for single events. The Pro upgrade is feature-gated rather than addon-gated.

### 4.3 Eventin (Free vs Pro)

**Eventin Free (WordPress.org, v4.1.0+):**
- Unlimited events
- Online & offline events
- Event attendee registration
- Recurring events (YES — rare for free tier)
- Multi-tier ticket levels
- WooCommerce payment support
- Calendar theming
- Custom fields

**Eventin Pro ($79/yr+) adds:**
- AI-powered event generation
- Template builder (pages, tickets, certificates)
- Email automation builder
- QR code check-in
- Event CRM & attendee management
- Front-end event submission
- Advanced payment gateways (Stripe, PayPal non-WooCommerce)
- Speaker & organizer management

**Key insight:** Eventin's free version is the most generous of the three P0 competitors — it includes recurring events and multi-tier tickets that are Pro-only in TEC and addon-only in EventON. The Pro version gates AI features and advanced tools, not core event management.

## 9. Dateline Takeaways

### What to Emulate

1. **EventON's loss-leader core pricing strategy.** A low one-time cost for the core plugin ($40) creates minimal purchase friction. The real monetization comes from add-ons. For Dateline, this maps well to a freemium model where the basic EmDash plugin is affordable/free and premium features (ticketing, seating, RSVP) are paid modules.

2. **Eventin's "everything included" simplicity.** The most successful pricing model in this analysis is Eventin Pro's: one price, all features, no add-on complexity. Users hate nickel-and-diming. For Dateline, consider whether an "all-in" tier with a higher price point reduces support burden and increases satisfaction.

3. **Bundle discounts (15–30%).** EventON and TEC Pro both use bundles to increase ARPU. Cross-selling complementary features (ticketing → seating → QR check-in) creates a natural upsell path.

4. **Lifetime license option.** Eventin Pro's lifetime pricing is a powerful competitive weapon. Even at a high price point ($299–449), lifetime access removes renewal anxiety. For Dateline, consider a lifetime tier for early adopter acquisition.

5. **Generous free tier.** Eventin's free version includes recurring events and multi-tier tickets — features that are Pro-only in TEC. This drives conversion because users outgrow the free version organically rather than hitting artificial gates.

6. **Clear pricing page.** Eventin has the most transparent pricing page. TEC Pro hides individual product prices behind bundles. EventON lists addon prices clearly. Transparency builds trust.

### What to Avoid

1. **EventON's addon fatigue.** Managing 30+ individual addon subscriptions creates cognitive overhead and cancellation risk. Users report confusion about what they need. Fewer, more integrated modules are better.

2. **TEC Pro's price creep.** The February 2026 increase (+33% on 1-site) generated significant negative sentiment. If raising prices, do so with clear communication and grandfather existing customers.

3. **No published refund policy (TEC Pro).** In a competitive market, an explicit refund guarantee (15–30 days) is table stakes. Not publishing one is a trust deficit.

4. **Annual-only subscriptions (TEC Pro).** With no lifetime option, TEC Pro's pricing feels like renting. Both EventON (core) and Eventin (lifetime tiers) offer buy-once options.

5. **Third-party gating (EventON's WooCommerce dependency).** Requiring WooCommerce for ticket sales adds complexity and friction. Native payment integration (like Eventin's Stripe/PayPal) is preferred.

### Total Cost of Ownership Comparison (1-site, 3-year horizon)

| Scenario | EventON | TEC Pro | Eventin Pro |
|----------|---------|---------|-------------|
| Basic calendar | $40 (core) | $597 (3yr renewal) | $79 (annual) or $99 (lifetime) |
| With ticketing | $40 + $120/yr × 2yr = **$280** | $199/yr × 3 = **$597** | $79/yr or **$99 lifetime** |
| Full-featured (tickets + RSVP + seats) | $40 + $450/yr (bundle) × 2yr = **$940** | $699/yr (bundle) × 3 = **$2,097** | $149/yr or **$299 lifetime** |

**Bottom line:** Eventin Pro offers the lowest TCO across all scenarios. EventON has the lowest entry cost but add-on subscriptions add up. TEC Pro is the most expensive across all time horizons. Dateline should target pricing between EventON and Eventin Pro — undercutting TEC Pro while maintaining perceived value.
